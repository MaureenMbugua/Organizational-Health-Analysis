# Organizational Health Analysis

## Project Overview

Designed and executed end-to-end data preparation and analysis for an organizational health assessment supporting strategic transformation decisions for a consulting firm's client. Transformed 50+ disparate survey questions into a structured analytical framework, enabling executive leadership to identify and prioritize transformation opportunities.

## Project Structure

- [Proposal](https://docs.google.com/presentation/d/1dNegY28WRu-Lf4iDCA1EaI9qMsGLhYkcYtjb67OmAxI/edit?usp=sharing): Analysis strategy and approach
- [data_preparation.ipynb](https://github.com/MaureenMbugua/Organizational-Health-Analysis/blob/main/Data_Preparation.ipynb): Detailed data preparation methodology
- [Organisational_Health_Dashboard.pdf](https://github.com/MaureenMbugua/Organizational-Health-Analysis/blob/main/Organisational_Health_Dashboard.pdf): Screenshots of the Power BI dashboard
- [Key-insights-recommendations](https://docs.google.com/presentation/d/1o_nymm-bjlkm8AFDf2gzfD5VMFHx0Cmh_DRsMNAa0yQ/edit?usp=sharing):Key insights and recommendations

**Note:** Actual survey data and client-identifying information are not included to maintain confidentiality.
