# Organizational Health Analysis

## Project Overview

Designed and executed end-to-end data preparation and analysis for an organizational health assessment supporting strategic transformation decisions for a consulting firm's client. Transformed 50+ disparate survey questions into a structured analytical framework, enabling executive leadership to identify and prioritize transformation opportunities.
